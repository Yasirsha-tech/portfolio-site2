Yasirsha's Portfolio - Static Export Guide

This guide will help you convert your Replit portfolio project into a completely static website that can be hosted anywhere without server dependencies.

What's in this directory

In the static-build directory, you'll find:

  • portfolio-files/ - The complete source code for your portfolio
  • README.md - Detailed instructions for creating a static version
  • tailwind.config.js - Tailwind configuration for your project

Quick Start

  1. Create a new Vite React project on your local machine
  2. Install the required dependencies
  3. Copy the files from portfolio-files/ to your project
  4. Build and deploy to any static hosting service

Step 1: Create a new Vite project

npm create vite@latest my-portfolio -- --template react-ts
cd my-portfolio
npm install

Step 2: Install required dependencies

npm install tailwindcss postcss autoprefixer
npm install tailwindcss-animate
npm install remixicon

Step 3: Configure Tailwind CSS

Initialize Tailwind and copy our configuration:

npx tailwindcss init -p

Replace the contents of tailwind.config.js with the one provided in this directory.

Step 4: Copy portfolio files

Copy all files from the portfolio-files/ directory to the corresponding locations in your new project's src/ folder.

Step 5: Update HTML template

Update your index.html file to include the required fonts and metadata.

Step 6: Build and deploy

npm run build

This creates a dist/ folder with your static site ready for deployment to any static hosting provider.

Note: For more detailed instructions, refer to the README.md file in this directory.
